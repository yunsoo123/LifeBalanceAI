**최종 수정**: 2025-02-15 기준

## 1. 지금 뭐부터 하면 되나요?

### A. 바로 할 수 있는 것 (돈 안 듦)
1. **고객 문의 이메일 정하기**  
   `docs/legal/privacy-policy.md`, `terms-of-service.md`에서 "(운영 중인 고객 문의 이메일을 추가해 주세요)" 부분을 실제 이메일로 수정 (예: `support@...` 또는 Gmail).
2. **앱 한 번 더 켜보기**  
   `cd LifeBalanceAI` → `npx expo start` 또는 `npx expo start --web` 으로 동작 확인.
3. **(선택) 앱 안에서 법적 문서 링크**  
   프로필/설정 화면에 "개인정보처리방침", "이용약관" 링크 추가.  
   문서는 나중에 웹에 올리거나 GitHub Pages 등에 올려서 URL을 쓰면 됨.

### B. 스토어 배포 전에 할 것
4. **통신판매업 신고**  
   인앱 결제(유료 구독) 하려면 통신판매업 신고 필요. 신고 후 이용약관/앱에 신고번호 기재.
5. **EAS Build로 앱 빌드**  
   iOS/Android 설치 파일(.ipa, .apk 등) 만들기.  
   - **무료**: EAS 무료 플랜으로 월 일정 횟수 빌드 가능.  
   - **유료**: 빌드 많이 하면 EAS 유료 플랜 (아래 "비용" 참고).
6. **스토어 등록**  
   - **Apple App Store**: 개발자 등록비 **연 $99 (약 13만 원)**.  
   - **Google Play**: 등록비 **일회 $25 (약 3만 3천 원)**.  
   앱스토어/플레이 콘솔에서 앱 생성 → 스크린샷·설명·개인정보처리방침 URL 등 입력 후 제출.
7. **인앱 결제 상품 등록**  
   App Store Connect / Play Console에서 구독 상품 ID `mendly_pro_monthly` 생성 후, 실기기에서 결제·복원 테스트.

### C. 배포 직전(여유 있을 때)
8. **테스트 목록 채우기**  
   `docs/01-plan/test-list-deferred.md`에 적어둔 대로, Phase 0~마지막까지 테스트 항목을 배포 직전에 작성.
9. **브랜딩**  
   앱 아이콘·스플래시·스토어용 스크린샷 준비.

## 2. 배포하려면 돈 들어가요?

### 꼭 드는 비용 (스토어에 올리려면)
| 항목 | 대략 비용 | 비고 |
|------|-----------|------|
| **Apple 개발자 프로그램** | 연 **$99** (약 13만 원) | iOS 앱 올리려면 필수. 매년 갱신. |
| **Google Play 등록** | 일회 **$25** (약 3만 3천 원) | Android 앱 올리려면 필수. |

### 선택적으로 드는 비용
| 항목 | 대략 비용 | 비고 |
|------|-----------|------|
| **EAS Build (Expo)** | 무료 플랜: 월 제한된 빌드 수 | 빌드 많이 하면 유료 플랜 ($29/월 등). |
| **Supabase** | 무료 플랜 있음 | 트래픽·DB 용량 넘어가면 유료. |
| **OpenAI API** | 사용량만큼 | 소규모면 월 몇 달러 수준. |
| **도메인·호스팅** | 연 수천 원~ | 개인정보처리방침/이용약관 웹 페이지 올릴 때만. |

### 정리
- **iOS만** 올리면: 연 **$99** (Apple) + (EAS 무료로 빌드 가능).
- **Android만** 올리면: 일회 **$25** (Google) + (EAS 무료로 빌드 가능).
- **둘 다** 올리면: **연 $99 + 일회 $25** + 사용하는 서비스(Supabase/OpenAI 등) 사용량.

인앱 결제(Pro 구독) 수익은 Apple/Google이 약 15~30% 가져가고, 나머지는 사업자(와이에스 프로젝트) 수입입니다.

## 3. 추천 순서 (요약)

1. 법적 문서에 **고객 이메일** 넣기 (5분)  
2. **EAS Build** 한 번 돌려서 설치 파일 만들기 (무료 플랜으로 가능)  
3. **Apple $99** 또는 **Google $25** 중 하나 결제 후 해당 스토어에 앱 제출  
4. 통신판매업 신고 후 **이용약관에 신고번호** 추가  
5. 스토어에서 **인앱 구독 상품** `mendly_pro_monthly` 등록 후 실기기에서 결제 테스트  

더 궁금한 거 있으면 `docs/05-status/current-status-report.md`랑 이 파일 참고하시면 됩니다.
