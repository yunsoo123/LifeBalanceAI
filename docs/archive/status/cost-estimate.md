# 토큰 비용 & 월 유지보수 예상 비용

**기준**: 2025년 상반기, Mendly 앱(OpenAI + Supabase + Expo)  
**참고**: 요금제·가격은 서비스 정책에 따라 변경될 수 있음.

---

## 1. 토큰 비용 (OpenAI API)

### 사용 모델
- **gpt-4o-mini** — 일정 생성, 노트 파싱, 노트 요약(enhance), 주간 리뷰 인사이트  
- Whisper(음성→텍스트) — 현재 미연동("추가 예정" 상태)

### gpt-4o-mini 요금 (참고용, 공식 페이지 확인 권장)
| 구분 | 가격 (USD) |
|------|------------|
| Input | 약 $0.15 / 100만 토큰 |
| Output | 약 $0.60 / 100만 토큰 |

- 1회 요청당 대략: **입력 500~1,500 토큰 + 출력 300~1,500 토큰**  
- 1회당 **약 $0.0005~0.002** (0.05~0.2센트) 수준으로 가정 가능.

### 월별 예상 (사용량 기준)
| 사용 규모 | AI 호출 수/월 | 토큰 비용 (대략) |
|-----------|----------------|------------------|
| **개인/테스트** | 50~100회 | **$0.05~0.20** |
| **소규모(월 100~500회)** | 100~500회 | **$0.20~1** |
| **중간(월 1,000~3,000회)** | 1,000~3,000회 | **$1~6** |
| **많은 트래픽** | 1만 회 이상 | **$10~30+** (추정) |

- 무료 한도(일정 10회/월, 파싱 50회/월, 인사이트 20회/월)만 쓰면 **월 $0.1 미만**도 가능.  
- Pro 사용자가 늘어나면 호출 수·토큰 비용이 함께 증가.

## 2. 월 유지보수 예상 비용

### 필수/거의 필수 (스토어 출시 시)
| 항목 | 내용 | 월 비용 (대략) |
|------|------|----------------|
| **Apple 개발자** | iOS 앱 배포 시 | **연 $99** → 월 약 **$8.25** |
| **Google Play** | Android만 쓸 때 | **일회 $25** → 월 0 (갱신 없음) |
| **OpenAI API** | 사용량 기반 | **$0~10** (사용량에 따라, 소규모면 $0~2) |
| **Supabase** | Free 플랜 | **$0** (한도 내) |
| **도메인** | 개인정보처리방침 URL용 (선택) | **$0~2** (연 $10~25 나누면) |

### 선택 (트래픽·기능 늘어날 때)
| 항목 | 내용 | 월 비용 (대략) |
|------|------|----------------|
| **Supabase Pro** | DB 8GB, 백업, 더 많은 이그레스 | **$25/월** |
| **EAS Build 유료** | 빌드 많이 할 때 | **$29/월** 등 (플랜별) |
| **Sentry 등 모니터링** | 에러 추적 | Free 티어 있음, 유료 시 **$26/월~** |

### 월 유지보수 “대표 시나리오”
| 시나리오 | 가정 | 월 예상 (USD) |
|----------|------|----------------|
| **최소 (테스트/개인)** | iOS 없음, Supabase Free, OpenAI 소량 | **$0~2** |
| **iOS 1개 스토어** | Apple $99/연 + Supabase Free + OpenAI 소량 | **약 $8~12** |
| **iOS + Android** | Apple $99/연 + Google 일회 $25 + Supabase Free + OpenAI 중간 | **약 $10~15** |
| **본격 운영** | Apple + Supabase Pro + OpenAI 사용량 증가 | **약 $35~50+** |

(위 금액은 세금·환율 제외한 참고치.)

## 4. 요약

- **토큰 비용**: gpt-4o-mini 기준 1회 호출 **약 $0.0005~0.002**. 월 100회면 **약 $0.1~0.2**, 1,000회면 **약 $1~2** 수준.
- **월 유지보수**:  
  - **최소** (웹/Android만, Free 티어 위주): **$0~2/월**  
  - **iOS 출시 시**: **Apple 연 $99** 포함 시 **월 약 $8~12**  
  - **운영 확대 시**: Supabase Pro·EAS 유료 등 포함 **$35~50+/월** 가정 가능.

가격은 [OpenAI Pricing](https://openai.com/api/pricing/), [Supabase Pricing](https://supabase.com/pricing)에서 최신 정보 확인하는 것이 좋습니다.
