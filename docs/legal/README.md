## 배포 전에 추가할 내용

1. **고객 문의 이메일**  
   개인정보처리방침·이용약관 문의처에 **실제 운영 이메일 주소**를 넣어 주세요.  
   (예: `support@mendly.app`, `mendly.contact@gmail.com` 등)

2. **통신판매업 신고번호**  
   통신판매업 신고를 한 경우, 이용약관 또는 별도 판매 페이지에 **통신판매업 신고번호**를 기재하는 것이 좋습니다.

3. **시행일**  
   현재 시행일은 2025년 2월 15일로 되어 있습니다. 실제 서비스 오픈일과 맞추어 수정해 주세요.

4. **앱/웹 노출**  
   - 앱: 설정·프로필 화면에서 "개인정보처리방침", "이용약관" 링크 연결  
   - 스토어: App Store / Play Store 제출 시 개인정보처리방침 URL 필수  
   - 웹에 게시하는 경우 해당 URL을 스토어와 앱 링크에 사용하면 됩니다.

이후 정책 변경·서비스 확장 시 두 문서를 수정한 뒤 시행일과 변경 내용을 공지하면 됩니다.
